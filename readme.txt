Adrecords Affiliate Plugin
Connect your WordPress website to your affiliate account at Adrecord. The plugin will help you make use of a feature called Clean Links, as well as give you news about new advertisers.

== Description ==

The plugin will connect your affiliate account at Adrecord to get from there to your WordPress website. You will need an account at Adrecord, and you can [sign up for free here](https://www.adrecord.com/en/signup). Read more about the [terms of service](https://www.adrecord.com/en/affiliate-agreement) and [privacy policy](https://www.adrecord.com/en/gdpr).

= Features =

* Get news about new advertiser to work with, right in your WordPress dashboard.
* Easily make use of Clean links.

== Installation ==

1. Install Adrecord Affiliate Plugin either via the WordPress.org plugin directory, or by uploading the files to your server (in the `/wp-content/plugins/` directory).
2. Activate the plugin.
3. After activating the plugin, enter your Adrecord API-key under the plugins settings panel

== Frequently Asked Questions ==

= Do I need an affiliate account at Adrecord? =

Yes, you will need an affiliate account at Adrecord to find this plugin useful. [Sign up for free.](https://www.adrecord.com/en/signup)

= Where do I find my API-key? =

You will find and be able to generate your API-key in the [Adrecord dashboard.](https://www.adrecord.com/en/advanced/api)

= What is Clean Links? =

Clean links is a technique to link to an advertiser with "regular links", you can read more about how it works in the [Adrecord dashboard.](https://www.adrecord.com/en/advanced/cleanlinks)
